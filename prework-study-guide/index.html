HTML

  • Hypertext Markup Lanaguge, or HTML, gives the content as well as the structure and the meaning of each element on the page.
  • Elements that are included in HTML are text elements, image elements, header elements, and list elements.
  • Besides elements that are seen there are elements used to divide content on a page such as meta-related elements, content division elements, and semantic elements.
  • The head element contains information about the webpage.
  • Under the head element metadata is located which is data about the webpage.
  • The body element represents the visible content shown to the user.
  • The stlye element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The script element embeds client-side scripts in the HTML document.
  • The title element sets the title of the webpage.

CSS

  • Cascading Style Sheets or CSS is the langauge used to describe the presentation of a website that's created with HTML and how each element on the page is rendered.
  • Inline CSS is used when you want to style and element directly in HTML.
  • The internal style sheet contains CSS rules for the webpage in the head section of the HTML file and the rules only apply to that page.
  • An external CSS style sheet is when the CSS rules are kept in a sepreate file.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A stages changes.
  • git commit -m “” allows you to leave a comment with your changes.
  • git pull origin main allows you to pull the branch into main.
  • git push origin allows you to push the changes to your remote repository.
